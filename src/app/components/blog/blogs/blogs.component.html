Can be found at: https://github.com/getify/You-Dont-Know-JS. Guys, I can't say enough about this fantastic resource for anyone learning JS. Really helpful, you can get them all FREE, all 6 books. You can purchase these in Kindle, hardback or paperback versions. If you have the spare doe and would like a copy, I recommend going for it, splash out! As it's such a great and coherent resource to have in the arsenal and supporting the author who made these brilliant resources free just seems right!

  • Up & Going

The Last Man by Vince Flynn

Can be found at: https://www.vinceflynn.com/the-last-man. This is a great read, the entire Mitch Rapp (ex-cop now PI) series is fantastic. This is book #13. If you would like to read the series, start here: American Assassin (Mitch Rapp #1). Good Reads Review - American Assassin.
